## 🧭 What QingJuan does

QingJuan is a desktop app for people who collect and read novels on a Windows PC. It brings four common tasks into one place:

-抓取小说内容
-整理作品信息
-下载本地文件
-翻译后阅读

It fits a simple daily use case. You find a novel source, pull in the content, keep it organized, save it to your computer, and read it in one app.

## 📖 Main features

### 🧲 Grab novel content
Pull text from supported novel pages and keep it in one place. This helps when you want to save chapters without copying each one by hand.

### 🗂️ Sort and manage
Keep novel titles, chapter lists, and reading progress in order. This makes it easier to return to a story later.

### ⬇️ Download for offline use
Save novels to your PC so you can read without a live internet connection. This also helps keep your reading list local.

### 🌐 Translate and read
Use translation support when you read content in another language. You can keep the text and translation close together for easier review.

### 🪟 Desktop reading
Read in a clean Windows window with a layout that suits long-form text. A desktop layout helps when you read for a long time.
