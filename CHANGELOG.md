# Changelog

本文档记录青卷的重要功能更新与行为变更。

## [0.3.0] - 2026-04-12

### 新增

- 桌面端无边框窗口新增顶部拖拽区域，支持按住顶部空白区域移动整个窗口位置。
- 桌面端新增窗口四边与四角的缩放热区，支持无边框窗口的边缘拖拽缩放。
- 本地小说导入支持多选 `TXT / TEXT` 文件并批量导入，前端会按顺序逐个写入文库，并汇总成功/失败结果。

### 改进

- 将关闭、最小化、最大化/还原功能区固定到右上角，并整体缩小，避免挤占主内容布局。
- 桌面端窗口标题会随当前视图、书籍与阅读章节同步更新，让任务栏与窗口标题更清晰。
- 应用主题与阅读主题完成状态同步，更好地处理 `默认 / 护眼 / 夜间 / 跟随系统` 之间的映射关系。
- 将项目版本统一提升为 `0.3.0`，同步更新桌面配置、前端包信息与界面显示版本号。

### 修复

- 修复桌面端切换主题后只有滚动条变化、但主界面背景和面板未同步生效的问题。
- 修复无边框模式下右上角窗口控制区被当作普通布局元素导致的错位和主区域空白问题。
- 修复关闭、最小化、最大化/还原按钮在 Tauri 2 自定义窗口下无效的问题，并补齐对应 window ACL 权限。
- 修复设置页输入框适配、按钮配色和较窄桌面宽度下的布局回落问题。
- 修复页面拖拽滚动与桌面窗口拖动手势互相冲突的问题。

## [0.2.1] - 2026-03-25

### 清理

- 移除旧的外部 AI 漫画翻译命令链路，统一切换到后端内置图片处理流程。
- 删除设置页中不再使用的漫画 AI 命令、LaMa 相关配置与对应接口。
- 清理 sidecar 构建脚本、PyInstaller 规格文件与仅旧链路使用的依赖项。

### 新增

- 新增漫画工作流，支持将远程链接识别为 `漫画` 类型，并进入按话/按页的导入、下载与阅读流程。
- 新增 `18comic.vip` 与 `bikawebapp.com` 的漫画站点支持。
- 新增 `Grok2API` 翻译提供商配置入口。
- 新增 Bika 账号自动创建与本地凭证复用逻辑。

### 改进

- 阅读进度从“只记录章节”升级为“章节 + 章节内位置”的持久化方案。
- “继续阅读”会尽量恢复到上次退出时的段落或图片附近。
- 切换上下章时自动回到页面顶部开始阅读。
- 切换原文/译文阅读模式时尽量保持当前阅读位置。
- 已安装桌面版默认将数据库、书库、翻译结果与资源写入应用同级 `data/` 目录。

### 修复

- 修复 18comic 阅读阶段的章节图片修复链路，降低图片乱序与切片异常带来的阅读失败概率。
- 修复“继续阅读”恢复时机过早、无法跳到真实位置的问题。
- 修复桌面端非阅读页侧栏跟随页面一起滚动的问题。
- 修复侧栏收缩后 Logo 与折叠按钮重叠的 UI 问题。
- 修复漫画长文本译图溢出、旧版翻译缓存不失效与目标语言识别错误等问题。
